# Optimized_Program_1


 *Optimized Java Programs Collection*

This repository contains a collection of optimized Java programs covering core concepts of Java development, including:

- File Handling
- Multithreading
- Streams API
- Collections (Map, Grouping)
- Optional Handling
- Mini Project (Employee Log System)

📌 Features Covered

1. File Handling

- Write data to file ("students.txt")
- Read file data
- Append data to existing file
- CSV file reading

2. Multithreading

- Thread creation using:
  - Thread class
  - Runnable interface
- Thread sleep
- Multiple threads execution
- Thread synchronization (Bank Account example)

3. Java Streams API

- Sorting employees using streams
- Filtering and processing collections

4. Collections (Map & Grouping)

- Map operations
- Group students by marks

5. Optional Class

- Handling null values safely using "Optional"

6. Mini Project: Employee Log System

- Add employee
- Store employee data in file
- Log actions in "app.log"
- Basic CRUD operations
