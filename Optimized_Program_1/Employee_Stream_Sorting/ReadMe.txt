# Employee Stream Sorting – Java 8

Sort a list of Employee objects by salary using **Java 8 Stream API**.

---

## Files
- `Employee.java` – Employee class with ID, Name, Salary, getters, and `toString`  
- `EmployeeMain.java` – Main class demonstrating sorting with streams

---

## Features
- Stream API: `stream()`, `sorted()`, `forEach()`  
- Sorting: ascending & descending order  
- Method references: `System.out::println`  
- Encapsulation: private fields + getters

---

## How to Run
```bash
javac *.java
java Optimized_Program_2.EmployeeMain
